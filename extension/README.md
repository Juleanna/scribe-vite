Scribe Click Capture (Chrome Extension)

Назначение
- Делать скриншот активной вкладки по клику мышью на странице и передавать его во вкладку приложения Scribe (http://localhost:5173).

Как установить (режим разработчика)
1) Откройте Chrome → chrome://extensions.
2) Включите Developer mode (режим разработчика).
3) Нажмите Load unpacked и выберите папку `extension/` из проекта.

Использование
1) Откройте ваше приложение Scribe на http://localhost:5173 (должна быть открыта вкладка с приложением).
2) Нажмите на иконку расширения "Scribe Click Capture" — это переключает режим (на иконке появится бейдж ON).
3) Перейдите на любую вкладку/сайт, кликните мышкой — будет сделан скриншот видимой области вкладки.
4) Скриншот автоматически попадёт во вкладку Scribe как новый шаг.
   На изображении рисуется:
   - синяя рамка вокруг выбранного элемента (его область),
   - красная стрелка, указывающая на точку клика.

Примечания
- Расширение использует chrome.tabs.captureVisibleTab: скриншот создаётся из того, что видно на экране.
- Координаты клика сейчас не используются (зарезервировано для будущих подсветок/аннотаций).
- Приложение должно быть открыто на http://localhost:5173 (или 127.0.0.1:5173). Путь можно расширить в background.js (chrome.tabs.query).

Shortcut: Alt+C toggles capture mode across any Chrome tab/window. To change it, open chrome://extensions/shortcuts and set a custom shortcut for 'Toggle click capture mode'.
